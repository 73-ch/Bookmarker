重要度に応じたブックマーク

favorites

コマンド : "command + 1"

頻繁に使うブックマークを登録します。

check

コマンド : "command + 2"

あとで見るブックマークを登録します。一度見ると削除されます。

others

コマンド : "command + 3"

その他のブックマークを登録します。

コマンド : "command + 4"

プロジェクトにブックマークを登録します。プロジェクトについてはあとで説明します。

検索結果

検索結果は矢印キーで移動して、enterキーで開くことができます。

shiftキーを押しながらenterキーを押すことで、新しいタブで開きます。

viやemacsの"command / ctrl + n / p"での移動にも対応しています。

左クリック・右クリックでも現在のタブで開くか新しいタブで開くことができます。

プロジェクト

プロジェクトは、ウィンドウの中のタブを全て管理します。

ウィンドウを閉じるだけで、最後に開いていたタブを自動でプロジェクトのブックマークに登録します。

それによって、次に作業を再開するときには前回のタブが復元されます。

もちろん、タブを消せばウィンドウを閉じるときに自動的に削除されます。

戻る 次へ

プロジェクトを作成する方法

1. ウィンドウにプロジェクトに入れたいタブを全て開きます。

2. "command / ctrl + p" か　popup windowの中のプロジェクト作成ボタンを押す。

3. 名前を入力して、enterキーを押すとプロジェクトに登録されます。
